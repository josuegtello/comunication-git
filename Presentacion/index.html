Definición

La inteligencia corporal o kinestésica es la capacidad para usar todo el cuerpo para expresar ideas y sentimientos y la facilidad en el uso de las propias manos para producir o transformar cosas. Esta competencia incluye habilidades físicas como la coordinación motriz, el equilibrio, la destreza, la fuerza, la flexibilidad y la velocidad, así como las capacidades auto perceptivas, las táctiles y la percepción de medidas y volúmenes.

Definición

Las competencias o habilidades ejecutivas son precisamente esas habilidades cerebrales, esos procesos cognitivos que permiten la adaptación a situaciones nuevas y no rutinarias. Se trata de aptitudes indispensables para adaptarse a situaciones para las que no hay soluciones preparadas, aptitudes indispensables para planificar, organizar, elaborar estrategias, pero también para estar atento y recordar detalles.
